# Docker Static Website

This project is a static website deployed using Docker and Nginx.

## 🚀 Features
- Static website with HTML, CSS, and JS
- Dockerized using Nginx
- Portable and lightweight

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- Docker
- Nginx
